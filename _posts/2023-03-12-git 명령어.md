```bash
git config —global user.name “bandal07”

git config —global [user.email](http://user.email) “bandal0796@gmail.com”

git init - 새로운 저장소를 생성할 때 사용하는 명령어

git status - 현재 프로젝트의 명령어를 확인

git add . (마침표 = all) - “commit”을 쓰기 전까지 위에 있던 명령어들을 모아 놓는 것

git commit -m”메모” - 변화를 기록하는 것

git push origin master - github에다가 적용해주는 것

git clone "url 넣기" - 복사
```
